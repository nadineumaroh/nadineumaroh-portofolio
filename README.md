## Cara pakai
1. Taruh foto & sertifikat asli di folder `assets/` dengan nama file
   yang sama seperti di atas (jangan ganti nama, kecuali kamu juga
   ubah referensinya di index.html).
2. Export 3 PPT case study jadi PDF, taruh di folder `pdf/` dengan nama
   yang sama seperti di atas.
3. Folder ini (utuh, dengan struktur yang sama) tinggal di-upload ke
   GitHub Pages / Netlify / Vercel sebagai static site — tidak perlu
   server, karena ini cuma HTML+CSS+JS murni.

## Catatan
File README.txt di dalam tiap folder cuma panduan untuk kamu sendiri —
boleh dihapus sebelum upload final, tidak memengaruhi tampilan website.
